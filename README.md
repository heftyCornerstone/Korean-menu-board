## 한식 메뉴 렌더링 하기

### 구현 영상

### 요구사항
필수
1. 주어진 데이터를 확인합니다.
2. 배열 메서드로 데이터를 화면에 표시합니다.
   
선택사항
1. CSS로 간단한 스타일을 적용합니다.
2. 객체의 속성을 추가 해봅니다.(e.g. 가격, 카테고리 등)
3. `filter`를 사용하여 설명에 “고기”가 포함된 메뉴 항목만 화면에 노출시킵니다.

<br><br>

### 적용된 요구사항
필수
1. 주어진 데이터를 확인합니다.
2. 배열 메서드로 데이터를 화면에 표시합니다.

선택사항
1. CSS로 간단한 스타일을 적용합니다.

특이사항
1. `filter`를 사용하지 않았지만, 설명에 “고기”가 포함된 메뉴 항목만 화면에 노출시킬 수 있는 토글 버튼이 있습니다. 클래스 추가와 삭제를 이용하여 구현하였습니다.
